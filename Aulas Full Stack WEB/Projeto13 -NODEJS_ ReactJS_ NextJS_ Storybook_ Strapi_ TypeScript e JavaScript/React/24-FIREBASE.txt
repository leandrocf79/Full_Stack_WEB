A estrutura desse BD é o seguinte, pode criar vários colletions e em cada um os docs
Exemplo:
-post (collection)
	--123 (doc)
	---titulo: Aprender React
	---autor: LeandroCF

-user (collection)
	--1 (usuário 1) (doc)
	---nome: Leandro
	---idade: 18

	--2 (doc)
	---nome: User2
	---idade: 19


Replicar isso no BD.
