Escolha uma para para iniciar o projeto React

O nome desta pasta deve ser limpo, sem acentos.


Pode usar o terminal da IDE ou


*******Para abrir o terminal já dentro desta pasta digite cmd onde aparece o caminho a pasta.

Exemplo:

D:\GitHub\Full_Stack_WEB\React\01-primeira-aula-de react

Apague e simplesmente digite:
cmd
e entrer

** O terminal já estará com o endereço da pasta.


Iniciar um projeto com nome de "meuapp":
npx create-react-app meuapp

**Vão aparecer 2 comandos após instalar:
cd meuapp
yarn start ou  npm start

Pode apagar dentro de SRC todos os arquivos menos
App.js e index.js


Em index.js pode tirar os imports desnecessários, vai ficar assim:
---------------------

import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App';

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);

Em App.js:

export default function App(){
  return(
    <div>
      <h1>Bem vindos ao sistema</h1>
      <h2>&copy; leandrocf79@gmail.com</h2>
    </div>
  )
}



ou melhor:



import React from "react";

function App(){
  return(
    <div>
      <h1>Bem vindos ao sistema</h1>
      <h2>&copy; leandrocf79@gmail.com</h2>
    </div>
  )
}

export default App;
