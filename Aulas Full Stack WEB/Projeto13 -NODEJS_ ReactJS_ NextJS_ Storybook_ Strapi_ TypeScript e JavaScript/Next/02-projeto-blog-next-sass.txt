Para iniciar um projeto com NextJs
pode ser pelo npx ou yarn a diferença é que yarn é mais rápida a instalação.
 
Tem que instalar o instalador yarn antes com
npm install --global yarn

O comando Shift + Alt + F
 indentação automática

--------------------------
Verifique:
yarn --version


Formas de começar. 
yarn create-next-app NOME-DO-PROJETO --typescript

OU
 yarn create next-app .    Com ponto para usar a propria pasta que acessou como nome do projeto

-----

Pode instalar o TypeScrip como dependência de Desenvolvimento
yarn add typescript -D

Instalar as tipagens
yarn add @types/react @types/node -D

-----------------Criar src/components/ActiveLink/index.tsx:


import { ReactElement, cloneElement } from 'react';
import { useRouter } from 'next/router';
import Link, { LinkProps } from 'next/link';

interface ActiveLinkProps extends LinkProps{
  children: ReactElement;
  activeClassName: string;
}

export function ActiveLink({ children, activeClassName, ...rest }: ActiveLinkProps){
  const { asPath } = useRouter(); 

  const className = asPath === rest.href ? activeClassName : '';
  //Se a rota/pagina que estamos acessando for igual ao link que ele clicou entao ativamos o classname

  return(
    <Link {...rest} legacyBehavior>
      {cloneElement(children, {
        className
      })}
    </Link>
  )
}
